Summary/Grading:

Ask not where I used bootstrap, ask, where did I not use bootstrap. The answer is that question is the page you're looking at right now, but every other page currently accessible through the app is certainly bootstrap-ified.

  1. Glyphicons, you can find them in the nav and the alerts.
  2. Input groups, you can find them on the complete student registration page.
  3. Nav, my nav is a fixed to the top bootstrap-themed nav.
  4. Labels, used in displaying the student profile and other places.
  5. Page header, used as the header for most pages.
  6. Alerts, used in various places when forms are submitted.
  7. List groups, used in displaying student profile info.
  8. Panels, used to display chunks of information, most notably, I have nested panels in the DGS overview.
  9. Input group addon, used to prepend the u in front of a unid during student registration.
  10. Forms, all of my forms use form-horizontal which treats form-groups as rows.
  11. Extensive use of the grid system to organize chunks of information in a responsive manner, all pages are mobile friendly.

UI Choice:

I took out my css everywhere but the readme, because I feel like this page really isn't a part of the application. I made heavey use of panels, the grid system, and other bootstrap components to organize information.
